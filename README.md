# Курсовой проект по Программированию на языке Java НИЯУ МИФИ (Сервис коротких ссылок)

Проект Short Link Service посвящён разработке сервиса коротких ссылок. 
В текущей версии реализованы ключевые решения, которые отвечают за базовую конфигурацию домена,
контроль времени жизни и лимитов переходов, а также периодическую очистку просроченных ссылок.

---

## Возможности

- Создание короткой ссылки.
- Переход по короткой ссылке.
- Ограничение времени жизни и числа переходов по ссылкам.
- Периодическая очистка просроченных ссылок.
- Получение уведомлений при достижении лимита переходов или по истечению ttl.
- Удалить и редактировать короткую ссылку.

---

## Быстрый старт

### Запуск из IDE
1. Клонируйте проект.
2. Установите зависимости (Maven).
3. Откройте класс `ShortLinkApp` и запустите метод `main`.

### Конфигурация

Базовые параметры лежат в `src/main/resources/application.properties`:

- `domain` — домен для коротких ссылок.
- `ttlSeconds` — время жизни ссылки в секундах.
- `defaultMaxClick` — лимит переходов по ссылке по умолчанию.
- `cleanupSeconds` — период фоновой очистки просроченных ссылок.

## Архитектура и решения

- **Сборка и зависимости**: Maven, Java 21, JUnit 5 для тестов.
- **Форматирование**: Spotless с Palantir Java Format.
- **CI**: GitHub Actions выполняет сборку, тесты и `spotless:check`.
